# Release Notes (0.1.0)

## New Features
- Initial release of `poormanray` (`pmr`) CLI for managing EC2 clusters and distributing jobs.
- Core commands: `create`, `list`, `terminate`, `pause`, `resume`, `run`, `map` for full cluster lifecycle management.
- Built-in setup commands: `setup` (AWS credentials), `setup-d2tk`, `setup-dolma-python`, `setup-decon` for Dolma ecosystem provisioning.
- SSH key auto-import to EC2 from local `~/.ssh/` keys.
- Detached command execution via GNU screen with `-d/--detach`.
- `map` command for distributing scripts evenly across cluster instances.

## Changes
- Accepts any SSH key type, not just RSA.
