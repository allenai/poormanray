# Release Notes (1.5.0)

## New Features
- Added a `transfer` command to copy local files or directories to every selected cluster instance in parallel using repeated `-s/--source source:destination` arguments.

## Housekeeping
- Extracted shared CLI options (`--banner-timeout`, `-i`, `-k`, `-j`, `-u`, `-G`) into `_shared_instance_options()` to reduce duplication between `common_cli_options` and `transfer_cli_options`.
- `_default_ssh_key_path()` now reuses `DEFAULT_PRIVATE_KEY_NAMES` from `ssh_session.py` instead of maintaining a separate key list.
- `run_command` now raises `click.ClickException` instead of `ValueError` for non-running instances, consistent with other commands.
