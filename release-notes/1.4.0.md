# Release Notes (1.4.0)

## New Features
- Added a `transfer` command to copy local files or directories to every selected cluster instance in parallel using repeated `-s/--source source:destination` arguments.

## Changes
- AWS `Name` tag is now uppercase (matching AWS convention); other instance/bucket tags remain lowercase. GCP labels are unaffected.
- `update-cluster` and `update-bucket` now migrate legacy-cased AWS tag keys (e.g., lowercase `name` → `Name`, uppercase `Project` → `project`) and delete the old keys.

**Full Changelog**: https://github.com/allenai/poormanray/compare/v1.3.0...v1.4.0
