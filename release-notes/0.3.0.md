# Release Notes (0.3.0)

## New Features
- `pmr wait` command — Poll until all cluster instances are running and passing EC2 health checks. Supports optional readiness checks via `-c/--command` or `-s/--script`, configurable `--poll-interval` and `-T/--timeout`, with a spinner animation showing per-instance progress. Health checks run in parallel.
