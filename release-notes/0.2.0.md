# Release Notes (0.2.0)

## New Features
- `pmr ssh` command for interactive SSH sessions to cluster instances.
- Configurable instance username via `-u/--instance-username` (previously hardcoded to `ec2-user`).

## Changes
- D2TK setup script now handles single NVMe drive (direct mount instead of requiring RAID).
- D2TK setup updated to install Python 3.12, uv, GitHub CLI, and use `duplodocus` (nee `minhash-rs`).
- Refactored `Session` and `SessionContent` classes into separate `session.py` module.


**Full Changelog**: https://github.com/allenai/poormanray/compare/v0.1.0...v0.2.0
