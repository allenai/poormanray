# Release Notes (0.5.0)

## New Features

- Added `create_bucket`, `update_bucket`, and `delete_bucket` commands to manage S3 buckets directly from `pmr`.
- Added `update_cluster` to backfill missing cluster tags on EC2 instances without overwriting existing values.

## Housekeeping

- Added shared S3 helper logic in `aws_instance.py` for bucket lifecycle defaults, tag serialization, and non-destructive updates.

**Full Changelog**: https://github.com/allenai/poormanray/compare/v0.4.0...v0.5.0
