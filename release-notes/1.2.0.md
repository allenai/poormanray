# Release Notes (1.2.0)

## New Features
- Add `--banner-timeout` CLI option (default: 15s) to control the SSH banner timeout when connecting to instances.
- Additional logging with `PMR_LOG_LEVEL=debug` environment variable.
- Apply shortcut for small scripts: if all commands are one-liners, do not upload a script file, but run them directly via a single command.

## Changes
- The `wait --script` option is no longer supported and will raise `NotImplementedError`. Use `wait --command` instead.
- `--parallelism` is now properly passed through to all setup commands (`setup`, `setup-d2tk`, `setup-dolma-python`, `setup-decon`).
- Screen session names now use `poormanray-*` prefix instead of `olmo-cookbook-*`.

## Fixes
- Fix "Error reading SSH protocol banner" when connecting to many instances in parallel by adding configurable `banner_timeout` and retry logic (5 attempts with jitter) to `Session.connect()`.
- Setup commands no longer require base64 encoding for script transfer.
- Wrapper script generation uses absolute paths (`$SCRIPT_DIR`) instead of relative paths, fixing issues when scripts are run from a different working directory.

## Housekeeping
- Remove unused `script_to_command` utility from `utils.py`.
