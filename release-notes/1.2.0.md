# Release Notes (1.2.0)

## New Features
- Add `--banner-timeout` CLI option (default: 15s) to control the SSH banner timeout when connecting to instances.
- Additional logging with `PMR_LOG_LEVEL=debug`

## Fixes
- Fix "Error reading SSH protocol banner" when connecting to many instances in parallel by adding configurable `banner_timeout` and retry logic (5 attempts with 5s delay) to `Session.connect()`.
- Setup commands do not require base64 encoding.
- Fixed typing
