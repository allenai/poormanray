# Release Notes (0.4.0)

## New Features
- `pmr ssh` command — Open an interactive SSH session to a cluster instance, with an interactive selection menu when multiple instances match.
- `pmr version` command — Print the installed poormanray version.

## Fixes
- Fixed path typo in publish workflow (`poorman` -> `poormanray` in version bump step).

## Housekeeping
- Updated README: documented `wait`, `ssh`, and `version` commands; added missing `io2e` storage type; expanded common options table with 8 previously undocumented flags; improved examples.
- Cleaned up `cli.py`: rewrote module docstring to list all 14 commands; fixed 7 function docstrings with incorrect/missing parameters; removed ~50 redundant inline comments.

**Full Changelog**: https://github.com/allenai/poormanray/compare/v0.3.0...v0.4.0
